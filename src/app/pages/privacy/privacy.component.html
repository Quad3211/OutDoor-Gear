4. Security & HTTPS

All pages on this site are served over HTTPS using TLS 1.3. TLS (Transport Layer Security) encrypts the data between your browser and our servers, preventing interception by third parties. You can verify this by checking for the padlock 🔒 icon in your browser's address bar.

Key points about our security approach:

  • All form data is transmitted over encrypted HTTPS connections.
  • We implement client-side input validation to prevent malformed data submission.
  • No user-supplied data is rendered as raw HTML (output escaping prevents XSS).
  • We do not include inline JavaScript that processes user data.
  • Strict Content Security Policy (CSP) headers are applied on deployment.
